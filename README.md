# MayaFlux Homebrew Tap

## Installation

```bash
# Add this tap
brew tap mayaflux/mayaflux

# Install MayaFlux (development version)
brew install mayaflux-dev
```

## What's Included

`mayaflux-dev`: Development version of MayaFlux library
